3.2 Random Variables

A variable that take on different values randomly.

  • be coupled with a probability distribution that specifies how likely each of these states are.
  • discrete or continuous
    • countably infinite number也是discrete
  • x\text{x}x is the random variable; $ x_1, x_2$ are its possible value.

3.3 Probability Distributions

A probability distributions is a description of how likely a random variable or set of random variables is to take on each of its possible states.

Probability Mass Functions (PMF)

  • A probability distribution over discrete variables.
  • P(x=x)P(\text{x}=x)P(x=x); x∼P(x)\text{x}\sim P(\text{x})x∼P(x)
  • joint probability distribution: P(x=x,y=y)P(\text{x}=x, \text{y}=y)P(x=x,y=y) or P(x,y)P(x, y)P(x,y)
  • to be a PMF, a function PPP must satisfy the following properties:
    • The domain of PPP must be the set of all possible states of x.
    • ∀x∈x,0≤P(x)≤1\forall x \in \text{x}, 0 \leq P(x) \leq 1∀x∈x,0≤P(x)≤1
    • ∑x∈xP(x)=1\sum_{x \in \text{x}}P(x)=1∑x∈x​P(x)=1

Probability Density Functions (PDF)

  • to be a PDF, a function ppp must satisfy the following properties:
    • The domain of ppp must be the set of all possible states of x.
    • ∀x∈x,p(x)≥0\forall x \in \text{x}, p(x) \geq 0∀x∈x,p(x)≥0
    • ∫P(x)dx=1\int P(x)dx=1∫P(x)dx=1

3.4 Marginal Probability

  • ∀x∈x,P(x=x)=∑yP(x=x,y=y)\forall x \in \text{x}, P(\text{x}=x)=\sum_yP(\text{x}=x,\text{y}=y)∀x∈x,P(x=x)=∑y​P(x=x,y=y)
  • p(x)=∫P(x,y)dyp(x)=\int P(x,y)dyp(x)=∫P(x,y)dy

3.5 Conditional Probability

The probability of some event, given that some other event has happened.

  • P(y=y∣x=x)=P(y=y,x=x)P(x=x)P(\text{y}=y \vert \text{x}=x) = \frac{P(\text{y}=y, \text{x}=x)}{P(\text{x}=x)}P(y=y∣x=x)=P(x=x)P(y=y,x=x)​

3.6 Chain Rule of Conditional Probabilities

Chain rule, or product rule of probability. It follows from the definition of conditional probability.

  • P(x1,...,xn)=P(x1)∏i=2nP(xi∣x1,...,xi−1)P(\text{x}^1,...,\text{x}^n)=P(\text{x}^1)\prod_{i=2}^nP(\text{x}^i \vert \text{x}^1,...,\text{x}^{i-1})P(x1,...,xn)=P(x1)∏i=2n​P(xi∣x1,...,xi−1)

3.7 Independence and Conditional Independence

  • Two random variables x\text{x}x and y\text{y}y are independent if P(x,y)=P(x)P(y)P(x,y) = P(x)P(y)P(x,y)=P(x)P(y)
  • Two random variables x\text{x}x and y\text{y}y are conditional independent if P(x,y∣z)=P(x∣z)P(y∣z)P(x,y \vert z) = P(x\vert z)P(y\vert z)P(x,y∣z)=P(x∣z)P(y∣z)

3.8 Expectation, Variance and Covariance

  • the expectation of function f(x)f(x)f(x) w.r.t. P(x)P(x)P(x) is
    • Ex∼P[f(x)]=∑xP(x)f(x)\mathbb{E}_{\text{x} \sim P}[f(x)]=\sum_xP(x)f(x)Ex∼P​[f(x)]=∑x​P(x)f(x)
    • Ex∼p[f(x)]=∫p(x)f(x)dx\mathbb{E}_{\text{x} \sim p}[f(x)]=\int p(x)f(x)dxEx∼p​[f(x)]=∫p(x)f(x)dx
  • here we can also think random variable is also a function
    • E[x]=∑P(x)x\mathbb{E}[\text{x}]=\sum P(x)xE[x]=∑P(x)x
    • E[x]=∫p(x)xdx\mathbb{E}[\text{x}]=\int p(x) x dxE[x]=∫p(x)xdx
  • The linearity of expectation
    • E[αf(x)+βg(x)]=αE[f(x)]+βE[g(x)]\mathbb{E}[\alpha f(x)+ \beta g(x)]=\alpha \mathbb{E}[f(x)]+\beta\mathbb{E}\mathbb[g(x)]E[αf(x)+βg(x)]=αE[f(x)]+βE[g(x)]
  • variance
    • Var(f(x))=E[(f(x)−E[f(x)])2]\text{Var} (f(x)) = \mathbb{E}[ (f(x)-\mathbb{E}[f(x)])^2]Var(f(x))=E[(f(x)−E[f(x)])2]
    • Var(x)=E[(x−E[x])2]=E[x2]−E[x]2\text{Var} (\text{x}) = \mathbb{E}[ (\text{x}-\mathbb{E}[\text{x}])^2]=\mathbb{E}[\text{x}^2]-\mathbb{E}[\text{x}]^2Var(x)=E[(x−E[x])2]=E[x2]−E[x]2
  • covariance: how much two values are linearly related to each other
    • Cov(f(x),g(x))=E[(f(x)−E[f(x)])(g(y)−E[g(y)])]\text{Cov}(f(x), g(x))=\mathbb{E}[(f(x)-\mathbb{E}[f(x)])(g(y)-\mathbb{E}[g(y)])]Cov(f(x),g(x))=E[(f(x)−E[f(x)])(g(y)−E[g(y)])]
