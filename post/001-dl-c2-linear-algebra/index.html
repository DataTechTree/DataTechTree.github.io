2.3 Identify and Inverse Matrices

Identity Matrix

  • Defination: An identity matrix is a matrix that does not change any vector when we multiply that vector by that matrix.
    • ∀x∈Rn,Inx=x\forall x \in \mathbb{R}^n, \mathbf{I}_n \mathbf{x} = \mathbf{x}∀x∈Rn,In​x=x
    • In∈Rn∗n\mathbf{I}_n \in \mathbb{R} ^{n*n}In​∈Rn∗n

Inverse Matrix

  • Defination: A is a n * n matrix, The matrix inverse of $\mathbf{A} $ is denoted as A−1\mathbf{A}^{-1}A−1, and it is defined as the matrix such that

A−1A=In\mathbf{A}^{-1} \mathbf{A} = \mathbf{I}_n A−1A=In​

  • a system of linear equation $\mathbf{A} \mathbf{x} =\mathbf{b} $ can be solved

x=A−1B\mathbf{x}=\mathbf{A}^{-1} \mathbf{B} x=A−1B

2.4 Linear Dependence and Span

  • Defination: the span of a set of vectors is the set of all points obtainable by linear combination of the original vectors.
  • Defination: A set of vectors is linearly independent if no vector in the set is a linear combination of the other vectors.
  • For the matrix to have an inverse, the matrix must be square and all the columns must be linearly independent.
    • right inverse AA−1=In\mathbf{A}\mathbf{A}^{-1} = \mathbf{I}_nAA−1=In​
    • for square matrices, the left inverse and right inverse are equal.
  • Defination: A square matrix with linearly dependent columns is known as singular.
